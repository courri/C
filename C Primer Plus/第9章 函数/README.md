> 通常情况下，可以把关于变量的两类信息传递给一个函数。
如果函数调用形式为：   
function(x);   
这时传递的是x的值。但是如果使用下面这种函数的调用形式：   
function(&x);   
那么会把x的地址传递给函数。   
使用函数进行数据计算等操作时，可以使用第一种调用形式。但是，如果需要改变调用函数的多个变量的值时，就需要使用第二种调用方式。
